## Overview

**CrystalDiskMark** is a free, open-source disk benchmark utility from **Crystal Dew World** that measures **sequential and random read/write performance** of HDDs, SSDs, NVMe drives, and USB storage. Used by IT administrators, hardware reviewers, system builders, and storage engineers worldwide, it is the de facto standard for comparing drive speeds, validating RAID performance, and documenting storage baselines before and after upgrades.

> 🏆 **Why CrystalDiskMark?** One-click benchmarking with Peak, Real World, and Demo modes. MIT-licensed, portable ZIP or MSI installer, command-line automation, and silent deployment for enterprise fleets. Pairs perfectly with **CrystalDiskInfo** for complete storage health + performance monitoring.

## How It Works

CrystalDiskMark writes test data to the selected drive and measures how fast it can be read back — reporting results in **MB/s** for sequential tests and **IOPS** for random 4K tests. Select your drive, choose a profile (Default, NVMe SSD, Peak Performance, Real World Performance), click **All**, and results appear in seconds. For automation, use the built-in **command-line mode** to script benchmarks across multiple endpoints.

## FAQ

<details>
<summary><strong>How do I download CrystalDiskMark?</strong></summary>
<br>
Download <strong>CrystalDiskMark.msi</strong> from the <a href="https://github.com/CrystalDiskMark/download/releases/latest/download/CrystalDiskMark.msi">latest release</a> or browse all versions on the <a href="https://github.com/CrystalDiskMark/download/releases">Releases</a> page.
</details>

<details>
<summary><strong>Is CrystalDiskMark free for business use?</strong></summary>
<br>
Yes. CrystalDiskMark is released under the <strong>MIT License</strong> — free for personal, commercial, and enterprise use with no restrictions.
</details>

<details>
<summary><strong>How do I install CrystalDiskMark silently?</strong></summary>
<br>
Run <code>msiexec /i CrystalDiskMark.msi /qn /norestart</code> or use the Inno Setup flags: <code>/VERYSILENT /SUPPRESSMSGBOXES /NORESTART /SP-</code>
</details>

<details>
<summary><strong>What is the difference between Standard, Aoi, and Shizuku editions?</strong></summary>
<br>
All editions share the same benchmark engine. <strong>Standard</strong> has the classic UI. <strong>Aoi</strong> and <strong>Shizuku</strong> editions offer alternative visual themes. This MSI packages the Standard Edition.
</details>

<details>
<summary><strong>How do I benchmark an NVMe SSD?</strong></summary>
<br>
Select the NVMe drive, choose the <strong>NVMe SSD</strong> profile from the settings menu, and click <strong>All</strong>. The NVMe profile uses optimized queue depth and test size for accurate high-speed results.
</details>

<details>
<summary><strong>Can I run benchmarks from the command line?</strong></summary>
<br>
Yes. CrystalDiskMark supports command-line execution for scripted and automated benchmarking. See the official documentation for CLI parameters.
</details>

<details>
<summary><strong>Does CrystalDiskMark work with CrystalDiskInfo?</strong></summary>
<br>
Yes. <strong>CrystalDiskInfo</strong> monitors S.M.A.R.T. health and temperature; <strong>CrystalDiskMark</strong> measures read/write performance. Together they provide complete storage diagnostics.
</details>

<details>
<summary><strong>Why are my benchmark results lower than advertised specs?</strong></summary>
<br>
Drive speed depends on test size, queue depth, data pattern, system load, and thermal throttling. Use the <strong>NVMe SSD</strong> or <strong>Peak Performance</strong> profile, close background apps, and ensure the drive is not nearly full for best results.
</details>
